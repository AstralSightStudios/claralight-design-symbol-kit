# @claralight-design/symbol-kit-compiler

ClaraLight Design Symbol Kit 的 SVG 编译器，提供解析、语义分类、几何处理与多字重、多样式 SVG 生成能力。

## 安装

```bash
pnpm add @claralight-design/symbol-kit-compiler
```

## 使用

```ts
import {
  generateFigmaSvgSet,
  type CompilerConfigInput
} from "@claralight-design/symbol-kit-compiler";
```

完整配置与示例请查看 [Symbol Kit 仓库](https://github.com/AstralSightStudios/claralight-design-symbol-kit#readme)。
