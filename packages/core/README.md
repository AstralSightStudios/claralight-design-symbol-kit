# @claralight-design/symbol-kit-core

ClaraLight Design Symbol Kit 的平台无关核心包，提供 Symbol IR、字重定义和 SVG 渲染能力。

## 安装

```bash
pnpm add @claralight-design/symbol-kit-core
```

## 使用

```ts
import {
  createSymbolRenderModel,
  renderSvg,
  SymbolWeight
} from "@claralight-design/symbol-kit-core";
```

完整文档请查看 [Symbol Kit 仓库](https://github.com/AstralSightStudios/claralight-design-symbol-kit#readme)。
